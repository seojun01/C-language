컴퓨터에 데이터를 저장하는 공간은 크게 두 부류로 나뉨.
컴퓨터를 종료하면 데이터가 날아가는 휘발성 메모리, 컴퓨터를 종료해도 데이터가 날아가지 않는 비휘발성 메모리.

휘발성 메모리 : 램 (RAM)
비휘발성 메모리 : 롬 (ROM), HDD ...

컴퓨터는 메모리의 매우 빠른 특성 때문에 대부분의 데이터들은 메모리에 보관해 놓고 작업을 하게됨.

컴퓨터의 한 개의 메모리 소자는 0 혹은 1의 값을 보관할 수 있음.
이 이진수 한 자리를 가리켜 비트(Bit)라고 함.
이렇게 8개의 비트를 묶어서 바이트(Byte)라고 부름

8비트로, 1바이트로 나타낼 수 있는 수, 8자리 이진수로 나타낼 수 있는 최대의 수는
00000000 ~ 11111111 = 0 ~ 255 = 0 ~ 0xFF

다음 단위로 워드 (WORD)라는 단위가 있음.
컴퓨터에서 연산을 담당하는 CPU에는 레지스터(register)라는 작은 메모리 공간이 있는데, 이곳에다가 값을 불러다 놓고 연산을 수행하게 됨.
CPU에서 연산을 수행하기 위해 잠시 써놓는 부븐을 레지스터.
이러한 레지스터의 크기는 컴퓨터 상에서 연산이 실행되는 최소 단위라고 볼 수 있고,
이 크기를 워드라고 부름.
