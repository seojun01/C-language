기수법 : 수를 표현하는 방법

십진법 : 10개의 숫자를 이용함
253 = 2 x 10^2 + 5 x 10^1 + 3 x 10^0

십진수를 이진수로 변환하는 법 : 2로 나누는 연산을 반복해주면 됨

하지만, 이진수는 흔히 수가 너무 커지는 경향이 있음.
따라서, 프로그래머들은 보통 16진법을 사용하고 있음.

16진법에는 10개의 숫자와 6개의 알파벳으로 이루어져 있음.
0, 1, 2, ... , 9, A, B, C, D, E, F

123 = 7 x 16 + 11 = 0x7B
이때, 0x는 16진수로 나타나있다는 것을 표현함.
